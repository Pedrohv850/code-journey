# Calculadora de Desconto

Uma aplicação simples e moderna desenvolvida com HTML, CSS e JavaScript para calcular o valor final de um produto após aplicar um desconto percentual.  
O projeto possui uma interface limpa e responsiva, com feedback visual para o usuário ao realizar o cálculo.

---

# Demonstração

Funcionalidade principal:
Informe o valor original e a porcentagem de desconto — o sistema mostrará:

-O valor original
-O percentual de desconto
-Quanto você economiza
-O valor final após o desconto

Também há um botão para "limpar os campos" e "redefinir a calculadora".

---

# Tecnologias Utilizadas

-HTML5: estrutura do conteúdo.
-CSS3: estilização moderna com transparência, sombras e animações.
-JavaScript (ES6): lógica para cálculo e interação dinâmica.
