# Calculadora de IMC

Uma aplicação simples e interativa para calcular o Índice de Massa Corporal (IMC), desenvolvida com HTML, CSS e JavaScript.  
O projeto apresenta um design limpo e moderno, ideal para praticar conceitos de lógica e estilização no front-end.

---

# O que é o IMC?

O IMC (Índice de Massa Corporal) é uma medida internacional usada para identificar se uma pessoa está com o peso ideal em relação à altura.  
A fórmula utilizada é:
IMC = peso / (altura \* altura)

---

# Funcionalidades

-Cálculo automático do IMC  
=Exibição da classificação (abaixo do peso, normal, sobrepeso, obesidade, etc.)  
-Mensagem de erro para valores inválidos  
-Interface intuitiva e responsiva

---

# Tecnologias Utilizadas

-HTML5: estrutura da página  
-CSS3: layout e design com transparência e gradiente  
-JavaScript (ES6): lógica para cálculo e exibição de resultados
