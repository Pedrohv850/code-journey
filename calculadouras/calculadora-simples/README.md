# Calculadora simples

Uma aplicação simples e interativa para calcular contas simples de multiplicação, adição, subtração e divisão, desenvolvida com HTML, CSS e JavaScript.  
O projeto apresenta um design limpo e moderno, ideal para praticar conceitos de lógica e estilização no front-end.

---

# Funcionalidades

-Calcular constas simples  
-Exibição do resultado  
-Não pode calcular sem ter um valor para ser calculado  
-Interface intuitiva e responsiva

---

# Tecnologias Utilizadas

-HTML5: estrutura da página  
-CSS3: layout e design com transparência e gradiente  
-JavaScript (ES6): lógica para cálculo e exibição de resultados
